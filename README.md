# EcoLogits (August 2025 Archive)

**NOTE:** This is not my original work. It is a replication of the version of EcoLogits that was published August 1, 2025, hash code `66e859538f73d194a4b4d763f7c5daabe6342082`, committed by `samuelrince`. 

This exists as a snapshot of the figures available on EcoLogits before its more recent updates. You can see the more recent version of EcoLogits here: [genai-impact/ecologits-calculator](https://huggingface.co/spaces/genai-impact/ecologits-calculator). 

**Technical Note:** You may need to manually install `ecologits==0.7.3` if you clone this repository locally. The current main branch defaults to more recent equations that lead to significantly lower estimates than those available in August 2025.
